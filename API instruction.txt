Запит поточної погоди у локації

Приклад URL з поясненням:
http://api.weatherapi.com/v1/current.json?key=API_KEY&q=LOCATION&aqi=no

Де API_KEY - це ваш ключ, який ви отримуєте після реєстрації
LOCATION - це місто, про яке ви хочете отримати інформацію
Параметр aqi може приймати 2 значення (yes/no) та якщо вказоно 'yes' - додатково повертає інформацію про якість повітря у місті.

Приклад URL з реальними параметрами:
http://api.weatherapi.com/v1/current.json?key=e968df4dfb6a147888a683622132403&q=London&aqi=no

Запит повертає JSON з двома об`єктами, location та current

Об'єкт location містить інформацію про місто, країну, дату та час

Об`єкт current містить інформацію про різні погодні параметри, такі як: температура (По Цельсію та Фаренгейту), об'єкт condition з коротким текстовим описом поточної погоди (Sunny, Rainy, Cloudly etc.), інформацію про швидкість та напрям вітру, а також багато іншої інформації.

Більш детально з тим, які параметри приходять після запиту, можна ознайомитись в Оглядачі API
Розшифровку параметрів можна подивитись в офіційній документації. Для поточної погоди можна скористатись наступним посиланням: https://www.weatherapi.com/docs/#apis-realtime





Запит прогнозу погоди у локації

Приклад URL з поясненням:
http://api.weatherapi.com/v1/forecast.json?key=API_KEY&q=LOCATION&days=NUMBER_OF_DAYS&aqi=no&alerts=no

Де NUMBER_OF_DAYS приймає ціле число кількості днів, для яких потрібний прогноз (починаючи від сьогодні). 
Зверніть увагу, що у безкоштовній версії API максимальна кількість днів для прогнозу складає 3 дні.

Параметр alerts може додатково надіслати дані про погодні попередження.

Приклад URL з реальними параметрами:
http://api.weatherapi.com/v1/forecast.json?key=e968df4dfb6a147888a683622132403&q=London&days=3&aqi=no&alerts=no

Запит повертає JSON з трьома об`єктами, location, current та forecast
Об'єкт forecast масив forecastday, в якому і забрані дані щодо прогнозу на локації

Більш детально з тим, які параметри приходять після запиту, можна ознайомитись в Оглядачі API
Розшифровку параметрів можна подивитись в офіційній документації. Для пронозу погоди можна скористатись наступним посиланням: https://www.weatherapi.com/docs/#apis-forecast
