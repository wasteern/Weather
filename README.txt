Шаблон цього проекту створений для студентів курсу Web Middle задля виконання завдань по Maker Time.

Проект містить базову структуру папок та файлів, яка використовувалась у навчальних матеріалах протягом року.

У файлі index.html вже є підключенн до стилів (style.css), локального скрипа (script.js).
А також, вже підключені FontAwesome.

Усю необхідну інформацію по створенню веб-сторінок та веб-додатків можна знайти у роздаткових матеріалах.

Номери тем, які можуть бути корисними:

№2. DOM. XML та JSON
№3. Основи асинхронних запитів (AJAX)
№8. AJAX в jQuery
